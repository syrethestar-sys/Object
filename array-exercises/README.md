# Array Exercises

Do them in order. Read the cheat sheet on top of each file, then fill every
`// your code here`. Uncomment the TEST lines and run to check. Every exercise
has **3 cases** to check — all three must pass.

```
node 1-basics.js
node 2-methods.js
node 3-database-movies.js
node 4-database-employees.js
node 5-database-products.js
node 6-database-fifa.js
```

1. **basics** — index, length, push/pop/shift/unshift, includes/indexOf
2. **methods** — map / filter / find / reduce / some / every / sort. These
   REPLACE writing a loop by hand: each one runs your function over every item.
3–5. **databases** — three big lists of objects (movies, employees, products).
   Same tools every time. Combine everything: objects, conditions, methods,
   and at the end grouping array-of-objects into an object. Boss levels.
